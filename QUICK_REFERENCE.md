## Admin Dashboard Features

| Section | What You Can Do |
|---------|-----------------|
| **Dashboard** | View analytics, stats, charts |
| **Clients** | Manage forest managers, add/edit/delete |
| **Projects** | Track woodland projects, assign teams |
| **Team** | Manage subcontractors, certifications |
| **Photos** | Upload project photos, organize gallery |
| **Content** | Edit website content (CMS) |
| **Contacts** | View contact form submissions |
| **Settings** | User profile, password, theme |
