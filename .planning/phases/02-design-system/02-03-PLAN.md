    Create five core UI components in src/components/ui/ with TypeScript and Tailwind:

    **Button.tsx:**
    - Variants: primary (bg-primary), secondary (bg-secondary), outline (border-primary)
    - Sizes: sm, md, lg (padding and text size)
    - Props: variant, size, children, className, ...rest (HTMLButtonElement props)
    - Use clsx or cn() utility for className merging (create in src/lib/utils.ts if needed)
    - Base styles: rounded, font-semibold, transition, hover states
    - Ensure WCAG AAA contrast on all variants

    **Card.tsx:**
    - Clean white background (bg-offwhite)
    - Subtle shadow (shadow-sm hover:shadow-md transition)
    - Rounded corners (rounded-lg)
    - Padding (p-6)
    - Props: children, className, ...rest (HTMLDivElement props)
    - Flexible composition via children

    **Container.tsx:**
    - Max width container (max-w-7xl mx-auto px-4 sm:px-6 lg:px-8)
    - Responsive padding for all screen sizes
    - Props: children, className, size variant (sm: max-w-4xl, md: max-w-5xl, lg: max-w-7xl, full: max-w-none)
    - Centers content on page

    **Heading.tsx:**
    - Semantic heading levels (as prop: h1, h2, h3, h4, h5, h6)
    - Visual styles (size prop: xl, lg, md, sm)
    - Decouples semantic from visual (e.g., h2 as="h1" for SEO)
    - Props: as (heading level), size (visual size), children, className
    - Base styles: font-bold, text-charcoal, leading-tight
    - Responsive text sizes (text-3xl md:text-5xl pattern)

    **Section.tsx:**
    - Full-width section wrapper
    - Background color variants (bg-offwhite default, bg-light, bg-primary)
    - Padding (py-16 md:py-24 for vertical spacing)
    - Contains Container by default
    - Props: children, className, background variant, ...rest (HTMLElement props)

    All components must:
    - Export as default
    - Include TypeScript interface for props
    - Support className override for Tailwind flexibility
    - Use design system colors and typography
    - Be accessible (proper ARIA attributes where needed)

    DO NOT use CSS modules or styled-components - Tailwind only.
    DO NOT add unnecessary complexity - keep components simple and composable.

    - Props: values (array of {title, description, icon (optional)}), columns (2 or 3), className
    - Layout: Grid (grid-cols-1 md:grid-cols-2 lg:grid-cols-3)
    - Each value in a Card component
    - Heading for each value title
    - Paragraph for description
    - Optional icon/emoji display if provided
    - Gap spacing (gap-6 md:gap-8)

    **CTASection.tsx:**
    - Prominent call-to-action section
    - Props: title (string), description (string), primaryCta ({text, href}), secondaryCta (optional {text, href}), className

    - Uses Container, Heading, Button components
