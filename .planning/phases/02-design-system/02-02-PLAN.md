<task type="auto">
  <name>Task 1: Configure Inter font with next/font/google</name>
  <files>src/app/layout.tsx</files>
  <action>
    Import Inter from next/font/google in layout.tsx:

    ```typescript
    import { Inter } from 'next/font/google'

    const inter = Inter({
      subsets: ['latin'],
      weight: ['400', '500', '600', '700'],
      variable: '--font-inter',
      display: 'swap',
    })
    ```

    Apply to html element via className:
    ```typescript
    <html lang="en" className={inter.variable}>
    ```

    Update globals.css to set Inter as default font:
    ```css
    body {
      font-family: var(--font-inter), system-ui, sans-serif;
    }
    ```

    Use next/font/google for automatic optimization and self-hosting (no Google CDN requests = better performance and privacy).
    Set display: 'swap' to show fallback font immediately while Inter loads (critical for rural connections).
  </action>
  <verify>grep "next/font/google" src/app/layout.tsx && grep "font-inter" src/app/layout.tsx && pnpm run build succeeds</verify>
  <done>Inter font imported, variable configured, applied to html element, build succeeds with font optimization</done>
</task>

<task type="auto">
  <name>Task 2: Establish typography utilities in Tailwind and globals</name>
  <files>tailwind.config.ts, src/app/globals.css</files>
  <action>
    Extend Tailwind config with typography customization:

    In tailwind.config.ts theme.extend:
    ```typescript
    fontFamily: {
      sans: ['var(--font-inter)', 'system-ui', 'sans-serif'],
    },
    lineHeight: {
      'tight': '1.25',
      'relaxed': '1.625',
    },
    ```

    In globals.css, add typography CSS custom properties for semantic usage:
    ```css
    :root {
      /* Type scale - matches Tailwind defaults but explicit for reference */
      --font-size-xs: 0.75rem;      /* 12px */
      --font-size-sm: 0.875rem;     /* 14px */
      --font-size-base: 1rem;       /* 16px */
      --font-size-lg: 1.125rem;     /* 18px */
      --font-size-xl: 1.25rem;      /* 20px */
      --font-size-2xl: 1.5rem;      /* 24px */
      --font-size-3xl: 1.875rem;    /* 30px */
      --font-size-4xl: 2.25rem;     /* 36px */
      --font-size-5xl: 3rem;        /* 48px */

      /* Semantic font weights */
      --font-regular: 400;
      --font-medium: 500;
      --font-semibold: 600;
      --font-bold: 700;
    }
    ```

    Add base typography styles for consistent rendering:
    ```css
    h1, h2, h3, h4, h5, h6 {
      font-weight: var(--font-bold);
      line-height: var(--line-height-tight);
      color: var(--color-charcoal);
    }

    p {
      line-height: var(--line-height-relaxed);
      color: var(--color-charcoal);
    }
    ```

    This gives us Tailwind utilities (text-xl, font-semibold, leading-tight) AND CSS custom properties for non-Tailwind contexts.
  </action>
  <verify>grep "fontFamily.*font-inter" tailwind.config.ts && grep "font-size-base" src/app/globals.css && pnpm run build succeeds</verify>
  <done>Tailwind config extended with Inter font family, globals.css has typography custom properties and base styles, build succeeds</done>
</task>
