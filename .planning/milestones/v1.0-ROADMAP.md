# Milestone v1.0: MVP

**Status:** ✅ SHIPPED 2026-02-08
**Phases:** 1-10
**Total Plans:** 22

## Overview

Build a professional marketing website for Acer Forestry Ltd that converts three distinct audiences: Forest Managers seeking contractors, potential clients exploring services, and subcontractors looking for opportunities. The journey starts with modern technical foundation (Next.js + TypeScript), establishes professional branding aligned with Scottish forestry values, builds three targeted conversion funnels, integrates educational content, adds polished animations, optimizes for SEO and rural connectivity, and deploys to production.

## Milestone Summary

**Key Decisions:**

- Next.js 15 over plain React - SEO-friendly, excellent Vercel integration, Framer Motion support
- Three-funnel architecture - Tailored messaging for Forest Managers, Services, and Subcontractors
- Professional + natural design blend - Credibility with Scottish landscape connection
- Soft launch strategy - Monitoring period for content enhancement before public announcement
- Title template system ('%s | Acer Forestry') - Consistent branding across pages
- LocalBusiness schema over Organization - Location-specific contractor with defined service areas

**Issues Resolved:**

- create-next-app directory conflict - Moved old site to backup, manual setup succeeded
- Port 3000 occupied - Next.js auto-selected port 3001, no blocker
- Testimonials unavailable - Design accommodates future addition when collected
- Training videos not available - Placeholder infrastructure created for future embedding

**Issues Deferred:**

- Custom domain configuration - acerforestry.co.uk DNS setup deferred to manual configuration when user ready
- High-quality service photos - To be captured on job sites and added during content enhancement phase
- Actual training video content - To be recorded/sourced during soft launch period
- Client testimonials - To be collected from satisfied Forest Managers

**Technical Debt Incurred:**

- None significant - clean codebase with consistent patterns established
